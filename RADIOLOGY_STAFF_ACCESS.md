# كيفية الوصول إلى صفحة طلبات الأشعة

## ✅ النتائج من الاختبار:
- ✓ الطلب #36 موجود في قاعدة البيانات
- ✓ حالته: `pending_service_selection`
- ✓ حالة الدفع: `not_applicable`
- ✓ يجب أن يظهر لموظف الأشعة

## 🔗 الروابط الصحيحة:

### للوصول إلى صفحة الطلبات:
```
http://127.0.0.1:8000/staff/requests
```

أو للأشعة فقط:
```
http://127.0.0.1:8000/staff/requests/radiology
```

## 👤 معلومات تسجيل الدخول:

**موظف الأشعة:**
- البريد: `radiology@hospital.com`
- كلمة المرور: (الكلمة المعتادة)

## 📋 خطوات الوصول:

1. **تسجيل الدخول** بحساب موظف الأشعة
2. **التوجه إلى**: `/staff/requests/radiology`
3. **يجب أن ترى**:
   - طلب #36
   - المريض: محمد أحمد علي
   - الحالة: بانتظار تحديد الخدمات

## 🔍 إذا لم يظهر الطلب:

### الحلول:
1. **مسح الكاش**:
   ```bash
   php artisan cache:clear
   php artisan view:clear
   php artisan route:clear
   ```

2. **تحديث الصفحة** (Ctrl + F5)

3. **التحقق من تسجيل الدخول**:
   - تأكد أنك مسجل دخول بحساب `radiology@hospital.com`
   - وليس بحساب آخر

4. **فتح الرابط المباشر**:
   ```
   http://127.0.0.1:8000/staff/requests/radiology
   ```

## 🎯 ما يجب أن يحدث:

عند فتح صفحة الطلبات، يجب أن ترى:

```
┌──────────────────────────────────────────────┐
│  إدارة الطلبات الطبية                      │
│                                              │
│  [تحديث]  [تصفية]                          │
│                                              │
│  ┌────────────────────────────────────┐     │
│  │ طلب #36                            │     │
│  │ النوع: أشعة                        │     │
│  │ المريض: محمد أحمد علي              │     │
│  │ الحالة: بانتظار تحديد الخدمات     │     │
│  │ [عرض التفاصيل]                     │     │
│  └────────────────────────────────────┘     │
└──────────────────────────────────────────────┘
```

## 🚀 الخطوة التالية:

بعد رؤية الطلب:
1. اضغط على "عرض التفاصيل"
2. ستظهر لك قائمة بأنواع الأشعة المتاحة
3. اختر الأشعة المطلوبة
4. اضغط "تأكيد الأشعة وإرسال للكاشير"
