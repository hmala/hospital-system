# 🖨️ نظام الأرشفة التلقائي - دليل الاستخدام

## ✨ ما الجديد؟
النظام الجديد **يعمل مع أي سكانر** متصل بـ Windows تلقائياً بدون الحاجة لـ NAPS2!

---

## 📋 المتطلبات

### 1. Python (مرة واحدة فقط)
- حمّل Python من: https://www.python.org/downloads/
- **مهم جداً:** فعّل خيار "Add Python to PATH" أثناء التثبيت ✓

### 2. سكانر متصل
- أي سكانر يدعم Windows (HP, Canon, Epson, Brother, إلخ)
- تأكد من تثبيت تعريفات السكانر

---

## 🚀 طريقة الاستخدام

### الخطوة 1: تشغيل برنامج السكانر
```
انقر مرتين على: تشغيل_السكانر_التلقائي.bat
```

سترى رسالة مثل:
```
╔═══════════════════════════════════════════════════════════════════╗
║  🖨️  نظام الأرشفة التلقائي - Hospital Auto Scanner Bridge v2.0  ║
╚═══════════════════════════════════════════════════════════════════╝

🔍 جاري البحث عن السكانرات المتصلة...
✓ تم العثور على 1 سكانر:
   1. HP LaserJet (HP)

🌐 الخادم يعمل على: http://localhost:37426
📡 جاهز لاستقبال طلبات المسح من الموقع
```

### الخطوة 2: المسح من الموقع
1. افتح صفحة إضافة عملية جراحية
2. ضع الورقة في السكانر
3. اضغط زر **"مسح الورقة من السكانر الآن"**
4. انتظر ثواني... تم! ✅

---

## 🔧 استكشاف الأخطاء

### "برنامج السكانر غير مشغل"
- تأكد من تشغيل `تشغيل_السكانر_التلقائي.bat`
- لا تغلق نافذة CMD السوداء

### "لم يتم العثور على سكانر"
1. تأكد من توصيل السكانر بالكمبيوتر
2. تأكد من تشغيل السكانر
3. تأكد من تثبيت تعريفات السكانر
4. جرب فصل وإعادة توصيل السكانر

### "فشل تثبيت المكتبات"
شغّل الأمر التالي في CMD:
```
pip install flask flask-cors pywin32 pillow
```

## 💡 نصائح

1. **أبقِ البرنامج مشغلاً** طوال فترة العمل
2. **الجودة الافتراضية** 200 DPI (مناسبة للوثائق)
3. **الصور تُضغط تلقائياً** لتوفير المساحة
